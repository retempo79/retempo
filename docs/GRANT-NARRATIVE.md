# Retempo Grant Narrative v0.1

## Project summary

Retempo is an agentic recurring payments and usage-based USDC settlement infrastructure project on Arc.

It helps developers, API providers, SaaS tools, and AI agents create payment agreements for subscriptions, API usage, and agent-to-service payments.

## Why Arc

Arc is used as the settlement layer for Retempo.

Retempo records final settlement proof on Arc and uses Arc transaction history as the source of truth for payment settlement.

## Why Circle

Circle Developer Platform is used as the wallet and transaction execution boundary.

The exact Circle integration path must be validated against official Circle documentation during implementation.

## Grant alignment

Retempo is aligned with:

- payments
- agentic commerce
- production-grade settlement infrastructure
- Arc-centered value flow
- USDC-based financial application infrastructure

## MVP scope

The MVP includes:

- developer dashboard
- service creation
- payment plan creation
- checkout sessions
- usage event records
- invoice state
- settlement records
- Arc transaction references

## Out of scope

Retempo v0.1 does not include:

- lending
- FX execution
- yield
- custodial balances
- payouts
- refunds
- disputes
- KYC
- tax handling
- multichain support

## Long-term direction

Retempo can evolve into a settlement infrastructure layer for AI agents and digital services that need programmable USDC payments on Arc.

# Retempo Product v0.1

## Positioning

Retempo is an agentic recurring payments and usage-based USDC settlement infrastructure project on Arc.

## Core decision

Retempo is a programmable recurring settlement layer for AI agents, SaaS APIs, and digital services.

It is not a lending protocol, FX product, prediction market, or regulated financial product.

## MVP focus

The MVP focuses on:

- agentic payments
- recurring USDC settlement
- usage-based API billing
- developer service plans
- checkout sessions
- invoice and settlement status

## Target users

- API providers
- AI agent builders
- SaaS developers
- digital service providers
- users or agents paying for services

## MVP scope

Included:

- landing page
- developer dashboard
- create service
- create payment plan
- checkout page
- usage event records
- invoice state
- settlement status
- Arc transaction history

Excluded:

- lending
- FX execution
- yield
- custodial balances
- payouts
- refunds
- disputes
- KYC
- tax handling
- multichain support
